## vue-test-rise

### Deployed URL
- Application
https://vue-test-rise.netlify.app/

- Storybook
https://rise-storybook-components.netlify.app/

### Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# run storybook
$ npm run storybook


```
